# consumir-api

# Consumindo uma API externa

Projeto simples feito para praticar o consumo de uma API externa usando JavaScript puro.

## O que o projeto faz

Busca uma lista de posts de uma API e exibe na tela com título, imagem e subtítulo.

## API utilizada

https://sujeitoprogramador.com/rn-api/?api=posts

## Tecnologias

- HTML
- CSS
- JavaScript (fetch, forEach, DOM)

## Como rodar

Baixe os arquivos e abra o `index.html` no navegador. Não precisa instalar nada.
